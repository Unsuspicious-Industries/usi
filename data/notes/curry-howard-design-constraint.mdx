If a program is a proof, then *generating* a well-typed program is *constructing* a proof. That reframing is the whole bet behind P7: don't ask the model to be correct, make ill-typed states unreachable.

The interesting residue is everything Curry-Howard *doesn't* fix — which branch of the proof to explore. That part is taste, and it stays in the model.
